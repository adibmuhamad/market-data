## Market Data

## Introduction

A simple market data service using Yahoo Finance as a data source.

## What you can find in this project

- Get Quote
- Get Quote Analyze, Recommendation Stock, Forecast
- Get Stock News Sentiment

## Prerequisites

- Go 1.16 or later
